# QR Code Generator using Python

## 📌 Project Overview
This project generates a QR code using Python. It allows users to create a QR code for any text or URL and optionally add a custom logo/image at the center of the QR code.

## 🛠️ Technologies Used
- Python
- Google Colab
- qrcode
- Pillow (PIL)

## ✨ Features
- Generate QR codes from text or URLs.
- Upload a custom image/logo from your computer.
- Place the uploaded image at the center of the QR code.
- Save and download the generated QR code.

## 📂 Output
The project successfully creates a QR code with a custom logo/image embedded in the center.

## 🚀 How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload your logo/image.
4. Run the code.
5. Download the generated QR code image.

## 📸 Result
A customized QR code is generated successfully with the uploaded image/logo.
